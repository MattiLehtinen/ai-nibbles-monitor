AI Nibbles Monitor
==================

A web based UI to watch games of [AI Nibbles server](https://github.com/MattiLehtinen/ai-nibbles-server)

Requirements
------------

Bower:
    
    npm install -g bower

Installation
------------

    git clone https://github.com/MattiLehtinen/ai-nibbles-monitor.git
    bower install
    
Running
-------

* Open `index.html` in your web browser
